# ProjetoFinal (Unity)

Jogo 3D estilo wave survival com zombies.

## Controlos
- WASD: mover
- Mouse: olhar / mirar
- Click esquerdo: disparar
- R: recarregar
- ESC: pause / opções

## Funcionalidades
- Sistema de rondas (waves) e spawn de zombies
- HUD (vida, munições, ronda)
- Sistema de save/load (SQLite)
- Menu (pausa, opções, som)

## Como abrir
Abrir a pasta do projeto no Unity Hub e carregar a cena em `Assets/Scenes/`.
